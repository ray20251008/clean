# 🧹 打掃工作流程製作器

這是一款專為**特殊教育班級**（如日間作業所、庇護工場）與**生活自理訓練**設計的智慧型步驟圖卡製作平台。

透過直覺的視覺化網頁介面，教師或訓練員可以輕鬆客製化各種清潔打掃流程，搭配**語音朗讀**與**互動打勾核取**功能，引導學生自主學習並掌握生活自理技能。

---

## 🌟 核心特色功能

1. **兩種使用模式切換**：
   - ✏️ **編輯模式**：點擊卡片即可編輯步驟名稱、說明、上傳實拍照片（支援手機直接照相、旋轉 90 度與自動壓縮防爆空間）。可直接在畫面上雙擊修改流程大標題與副標題。
   - 👁️ **預覽與實踐模式**：提供給學生使用。學生可以在平板電腦上，一邊打掃一邊將完成的步驟打勾，進度條會即時更新。當步驟全數完成時，會彈出可愛的「🎉 慶祝動畫」，並朗讀鼓勵語音，給予即時正向回饋。
2. **🔊 台灣語音合成朗讀 (TTS)**：
   - 每個步驟均配有「🔊 語音朗讀」按鈕。點擊後系統會以溫和清晰的台灣國語讀出步驟，讓識字困難或視覺學習型的學生能聽讀配合。
3. **🎨 自訂版面設計面板**：
   - **清新背景主題**：提供薄荷綠、天空藍、柑橘黃、薰衣草紫、櫻花粉與特教專用高反差黑黃配色。
   - **字型與長寬控制**：包含標題文字大小、步驟文字大小、圓角/方角卡片樣式、底部裝飾圖樣（清潔用品、水滴泡泡、或自訂上傳 5 張照片）、底部裝飾大小。
4. **🖨️ A4 放大列印與圖卡製作**：
   - 完美支援 `@media print` 樣式，可自訂「每頁列印 1/2/4/8 個步驟」。
   - 選擇「每頁 1 個步驟」即可列印出精美的大圖卡，方便護貝後貼在洗手台、黑板或垃圾桶旁做為環境視覺提示（Visual Supports）。
5. **📴 100% 離線可用，免安裝資料庫**：
   - 採用純 HTML5/CSS3/JavaScript 技術，無後端依賴。所有步驟與設計皆以 `LocalStorage` 自動儲存於本機瀏覽器中。

---

## 📁 檔案結構說明

- **`index.html`**：網頁主體架構與 UI 介面。
- **`style.css`**：視覺美化樣式、微動畫、黑白影印優化以及高精確列印分頁排版。
- **`app.js`**：核心控制邏輯（預設範本載入、TTS 朗讀、圖片旋轉/縮小壓縮、進度計算）。
- **`一鍵上傳到GitHub.bat`**：雙擊即可引導您將此工具發布到您個人的 GitHub Pages 線上使用。
- **`README.md`**：本說明文件。

---

## 🚀 快速開始指南

### 本地直接執行
1. 滑鼠連按兩下（雙擊）開啟 `index.html`。
2. 您可以直接在瀏覽器中開始使用。
3. 建議使用 Chrome 或 Edge 瀏覽器以獲得最穩定的語音朗讀（TTS）體驗。

### 內建四套預設範本
點擊網頁上方下拉選單，即可快速套用或切換：
- 🧹 **教室掃地與拖地**
- 🧽 **抹布擦桌子**
- ♻️ **倒垃圾與資源回收**
- 🧴 **洗手台清潔**

---

## ☁️ 發布至 GitHub Pages (線上化)
1. 在 GitHub 上建立一個新的公開 Repository，命名為 `cleaning-workflow`。
2. 雙擊執行資料夾中的 **`一鍵上傳到GitHub.bat`**。
3. 依照視窗提示，貼上您的 GitHub 倉庫網址並按 Enter，即可自動推送上傳。
4. 前往專案的 **Settings** -> **Pages**，將 Branch 設定為 `main` 分支並儲存。
5. 稍等一分鐘，即可在平板或手機上以專屬網址開啟此打掃流程圖！
